Data Science Intern

Link to full Warner Bros

This was a 10 week internship from June 2019 - August 2019

What was achieved

  • Developed Python image identification model for identifying any DC Universe characters in a given DC Universe comic page achieving 76% accuracy.
  • Presented project architecture and model’s layer activation to WB executive, data intelligence team, and DCU stakeholders.
  • Deployed Python web crawler that collects and parses DC comic images of any DC Universe Character.

How I achieved it

  • I built the model using Keras framework with Xception Convolutional Neural Network architecture.
  • I performed feature engineering, image augmentation, transfer learning, and hyper-parameter tuning on model located in AWS EC2 cloud instance.
  • I analyzed and tested model against different eras and art styles of Batman.
  • I visualize in the image did the model analyze using Keras’ Grad Cam Package
  • I set up a pipeline that automates the web crawler’s collection and storage of comic images.

Practical Uses

  • Batman’s location being highlighted on any given DC Universe comic page.
  • Once the base model of predicting Batman is achieved, use Transfer Learning to identify other characters.
  • Identify the different art styles and eras of Batman.

Note: Although this project was on Batman, I made the model general enough to utilize Transfer Learning to detect well-known characters such as; Joker, Lex Luthor, Wonder Woman, Superman and not as wellknown charcters such as Calendar Man, Granny Goodness and so on!

Challenges

  • Collecting images of Batman was difficult as there were some eras, issues and series that did not have enough images of Batman in their issues. This problem increased for other characters that did not already have a catalog of their own resulting in a smaller dataset.
  • Labeling images of superheros.
  • Loading and image augmentation of images.
  • Hyper-Parameter tuning.

Note: I built an IPython notebook at moves labeled images into their superhero or not folders, however this process took up most of my time since it had to be done manually. As the dataset grew larger, it became a bigger problem of loading raw and augmented images into the model so had to rebuild model into that dynamically augmented raw images as they were uploaded into the model.
