Riot Champion Trend and Item Recommender

Riot Champion Trends and Item Recommender is a data science project where I analyze and visualize a user’s performance in League of Legends. I compare their playing history data against a list of professional or similar ranking player’s game history data using R.

To obtain the list of professional players game data, I built a Python web crawler using the requests library and RIOT API to scrape their game data, storing them into JSON files. With these JSON files, I then conduct data analysis using R as I had no prior experience using the language.
